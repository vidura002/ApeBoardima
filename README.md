BoardingLanka is a modern room rental and boarding discovery platform for students and young professionals in Sri Lanka.It combines a responsive React + TypeScript frontend with a Node.js + Express backend, offering:

searchable, filterable listings
landlord and tenant dashboards
auth, saved rooms, and enquiries
property image uploads and location mapping
Built with:

Frontend: React 18, Vite, Tailwind CSS, React Router
Backend: Node.js, Express, Prisma
Database support: PostgreSQL / SQLite via Prisma

=======

> Premium room rental and boarding discovery for students and young professionals in Malabe, Sri Lanka.

## Stack

| Layer | Tech |
|---|---|
| Frontend | React 18 + TypeScript + Vite |
| Styling | Tailwind CSS 3 |
| Icons | Lucide React |
| Routing | React Router v6 |
| State | React Context API |
| Backend | Node.js + Express (optional, frontend is self-contained) |

## Quick Start

```bash
# Install frontend deps
cd client
npm install
npm run dev
# → http://localhost:5173
```

```bash
# Run backend API (optional)
cd server
npm install
npm run dev
# → http://localhost:3001
```

## Pages

| Route | Description |
|---|---|
| `/` | Home — hero, featured listings, areas, how it works |
| `/browse` | Browse & filter all listings |
| `/listing/:id` | Listing detail with image gallery, contact |
| `/auth` | Sign in / Sign up |
| `/dashboard/landlord` | Landlord dashboard — listings, inquiries, stats |
| `/dashboard/tenant` | Tenant dashboard — saved rooms, inquiries |
| `/create-listing` | Multi-step listing form for landlords |

## Demo Login

Enter **any email + any password** on the sign-in page to log in as a demo user.

To get a landlord demo account, use the "Create account" tab and select **Landlord**.
