# ApeBoardima
BoardingLanka is a modern room rental and boarding discovery platform for students and young professionals in Sri Lanka.It combines a responsive React + TypeScript frontend with a Node.js + Express backend, offering:

searchable, filterable listings
landlord and tenant dashboards
auth, saved rooms, and enquiries
property image uploads and location mapping
Built with:

Frontend: React 18, Vite, Tailwind CSS, React Router
Backend: Node.js, Express, Prisma
Database support: PostgreSQL / SQLite via Prisma
